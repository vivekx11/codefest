Event Details

Debugging Challenge

Description

Test your debugging skills by fixing code in multiple programming languages. Race against time to identify and resolve bugs in increasingly complex programs.

Languages Allowed

  • Python
  • JavaScript
  • Java
  • C++

Rules & Scoring

  • Time limit: 2 hours
  • Points awarded based on number of bugs fixed
  • Bonus points for optimal solutions
  • Penalties for introducing new bugs

Code in Dark

Description

Code without preview or syntax highlighting. Put your HTML and CSS skills to the test by coding a webpage without seeing the result until time's up.

Rules

  • 15 minutes per round
  • HTML and CSS only
  • No preview window
  • No syntax highlighting

Judging Criteria

  • Visual accuracy
  • Code quality
  • Responsive design
  • Completion time

UI/UX Design Challenge

Description

Design a modern user interface based on a given prompt. Showcase your creativity and understanding of user experience principles.

Tools Allowed

  • Figma
  • Adobe XD
  • Sketch

Evaluation Criteria

  • Visual design
  • User experience
  • Responsiveness
  • Innovation
